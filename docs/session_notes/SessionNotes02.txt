POSTMAN AUTOMATION (POST‑RESPONSE SCRIPTS)


Purpose:

Capture generated IDs
Auto‑increment business identifiers for demo flow

Used Post‑response script (new Postman UI terminology):
Script logic:

Runs when response code == 201
Parses JSON response
Extracts employee ID
Extracts numeric portion from employeeNumber string
Increments number
Saves both to collection variables

Working script:
if (pm.response.code === 201) {
const response = pm.response.json();
const match = response.employeeNumber.match(/(\D*)(\d+)/);
const prefix = match[1];
const number = parseInt(match[2], 10);
pm.collectionVariables.set("employeeId", response.id);
pm.collectionVariables.set("employeeNumber", prefix + (number + 1));
}
Result:

employeeId stored and reused automatically
employeeNumber incremented for next create
Manual copying avoided
